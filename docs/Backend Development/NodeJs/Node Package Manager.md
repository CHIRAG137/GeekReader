For using npm, we do not create js file rather we make a mjs file.
## Using Chalk Module
```
import chalk from 'chalk'
console.log(chalk.underline.blue.inverse.italic.bold.dim.hidden("hello"));
```

|underline| Underline |
| :--------: | :--------:|
|blue     | Underline |
|inverse  | Underline |
|italic   | Underline |
|bold     | Underline |
|dim      | Underline |
|hidden   | Underline |
